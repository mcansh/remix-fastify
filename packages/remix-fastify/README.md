# Remix Fastify

Use [Remix](https://remix.run) with [Fastify](http://fastify.io)

## Installation

```sh
npm i @mcansh/remix-fastify fastify
```

## Quick Start

```sh
npx create-remix@latest --template https://github.com/mcansh/remix-fastify/tree/main/example
```
